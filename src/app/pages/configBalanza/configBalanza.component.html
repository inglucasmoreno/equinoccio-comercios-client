Configuración de balanza

El formato de código de balanza es el formato que se utiliza para leer el código de barras de la balanza

Tipos de digitos:

di - Dígitos de iniciales

Pr - Dígitos de código de producto

Pe - Digítos de peso

df - Dígitos de finalizacion

Actualización de formato:

Para actualizar el formato se utiliza un string con expresiones reducidas

i - Dígitos de iniciales

p - Dígitos de código de producto

e - Digítos de peso

f - Dígitos de finalizacion

Ejemplo: iipppppeeeeef
